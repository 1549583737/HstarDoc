1、在构造函数中存储私有数据

该方式，和在ES5中，没有什么区别。

该方式有一个变种，就是利用构造参数来存储，减少重新定义变量。

  1. 数据绝对安全，外部无法直接通过属性访问到。

缺点：

  1. 代码不怎么优雅，需要把方法设置为实例方法，才能访问到私有数据。
  2. 实例方法，比较浪费内存（每个实例都会拷贝一份）。
