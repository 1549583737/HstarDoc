# 监控双向绑定

由于 $onChanges 无法监控双向绑定属性，那么我们就必须另外想办法来进行监控，可以有以下几种方案：

方案一：利用 $interval

既然是双向绑定，那么肯定变化是直接生效的，关键就在于我们无法监视到，这个时候我们可以利用 $interval 来实现定时监控。
