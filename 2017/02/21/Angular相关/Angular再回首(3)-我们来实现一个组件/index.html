2.4、组件属性绑定

之前实现的组件，感觉太死板了，我想改下文字，都不好实现（你非要用js强制操作dom，我拿你也没办法，不过后果自负），这个时候，我们迫切的需要能给组件传递参数。

Angular 组件中，有多个参数传递方式，如下：

  • @ 单向绑定字符串（原值绑定） - 传什么就是什么，不做任何处理
  • < 单向绑定变量（取scope的值绑定） - 传的值会先用 $scope 转换，把结果传递给组件
  • = 双向绑定 - 组件内外变化都会通知另一方
