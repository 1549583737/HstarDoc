1、组件生命周期

既然组件也是指令，那么指令所拥有的四大阶段组件也同样拥有。

而且，由于组件带有视图，还多了几个和视图相关的生命周期阶段。如下：

  1. ngAfterContentInit 组件内容渲染到页面之后触发
  2. ngAfterContentChecked 检查组件内容绑定数据后触发
  3. ngAfterViewInit 创建组件视图之后触发
  4. ngAfterViewChecked 检查组件视图绑定数据后触发

它们的执行顺序也和以上顺序一致。
