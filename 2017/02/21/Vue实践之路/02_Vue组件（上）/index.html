0、关于Vue组件

组件是 Vue 中最强大的功能之一，Vue组件也和angular的组件比较类似，可以扩展HTML元素。在较高层面上，也是自定义元素。

在原生HTML元素上附加功能，Vue 的做法是通过 is 特性扩展， ng2 中则称之为属性指令。
